\section{TThoriumPersistent}
\label{thoriumcore:thorium:tthoriumpersistent}
\index{TThoriumPersistent}
% Description
\subsection{Description}
You should derive as much as possible classes you want to publish to  Thorium from this base class. This class is abstract. You should never create an instance of this class, only of its descendants.\\
 The reference counter is mainly for the Thorium engine. So the complete host environment is counted as one reference. When you do not need the instance anymore, you should release it by \textit{one} call to FreeReference (not to Free, since there might be more references in the Thorium engine). % Method overview
